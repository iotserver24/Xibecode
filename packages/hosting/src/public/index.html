Your infrastructure

Each instance is a private E2B sandbox (4 cores / 8 GB) with XibeCode preinstalled.

1) Edit config

Update provider, model, base URL, API key, and Telegram bot token inside this instance. Leave secret fields blank to keep the current value.

Default is pairing on: when someone DMs the bot they get a code; approve it under option 3.

2) Restart daemon

Starts (or restarts) xibecode daemon in the sandbox using the config already saved there — no need to re-enter keys.

3) Pair Telegram

  1. Open Telegram and DM your bot any message.
  2. Bot replies with a pairing code like ABCD1234.
  3. Paste the code below and approve.
