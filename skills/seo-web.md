# SEO for Web Apps

## Metadata

- Unique titles and descriptions per route; canonical URLs when content duplicates.

## Structure

- Sensible heading hierarchy; meaningful link text; avoid blocking critical content behind client-only JS when SSR exists.

## Technical

- Sitemaps/robots when the product needs indexing; verify with search console patterns the team uses.
