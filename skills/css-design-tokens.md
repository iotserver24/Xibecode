# CSS & Design Tokens

## Consistency

- Reuse tokens/variables from the design system before introducing new colors or radii.

## Specificity

- Avoid deep selector wars; match the methodology in use (BEM, CSS Modules, Tailwind layers).

## Performance

- Prefer modern layout (flex/grid); avoid expensive selectors and huge global style sheets.
